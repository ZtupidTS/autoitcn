NNI接口
下列位置的 NNI 接口同SIP网络的路由、SIP 网络与 PSTN 的互通等问题密切相关，对
代理服务器、 B2BUA 和 SIP-ISUP互通单元的能力要求有着重要的影响， 故将其作为关键的
NNI 接口识别出来。涉及到的实体与协议主要包括：
1. SIP Proxy 之间：SIP或 SIP-I
2. SIP Proxy 与SIP-ISUP互通单元之间：SIP或 SIP-I
3. SIP-ISUP互通单元之间：SIP-I
4. SIP B2BUA之间：SIP 或SIP-I
5. SIP Proxy 与 SIP B2BUA之间：SIP 或 SIP-I
6. SIP-ISUP互通单元与SIP B2BUA之间： SIP 或SIP-I
本规范建议网络实体间建立信任关系，因而暂不要求 NNI 接口支持网络实体间的鉴权
和授权功能。
